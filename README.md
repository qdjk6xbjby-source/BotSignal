# Trading Signal Bot 🚀

Бот для Telegram, который анализирует рынок через TradingView и присылает высокоточные сигналы.

## Особенности
- **Таймфрейм 1М + 5М**: Анализ локального тренда и подтверждение на старшем таймфрейме.
- **Технические фильтры**: RSI (фильтр перекупленности) и ADX (фильтр силы тренда).
- **Pivot Points**: Контроль уровней поддержки и сопротивления.
- **Новостной фильтр**: Автоматическая блокировка сигналов за 30 минут до важных новостей.
- **Безопасность**: Работает только для разрешенных пользователей (`ALLOWED_USERS`).

## Установка на сервер
1. Склонируйте репозиторий.
2. Подготовьте файл `.env` (используйте `.env.example` как образец).
3. Запустите скрипт настройки:
```bash
chmod +x setup_server.sh
./setup_server.sh
```

## Запуск
```bash
source venv/bin/activate
python trading_bot.py
```

## Список валютных пар
Редактируется в `trading_bot.py` (переменная `SYMBOLS`).
